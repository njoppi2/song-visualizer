# Research Notes

Files in this folder are reference material, not execution defaults.

- `lyrics_syncing_research.md`: comparison of lyric transcription/alignment tools, caveats, and references.
- `voice-notes-research.md`: pitch extraction tool survey (pYIN, Basic Pitch, CREPE) for Phase 4 vocal note events.
- `song-story-research.md`: structural segmentation tool survey (SSM, MSAF, novelty detection).

For implementation decisions, start with `docs/05_lyrics_playbook.md`.
